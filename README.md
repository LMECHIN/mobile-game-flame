## :anger: Requirements ##

Before starting, you need to have [Flutter](https://flutter.dev/) installed.

## :heavy_check_mark: Build steps

```bash
git clone https://github.com/LMECHIN/game2d.git

cd game2d

flutter pub get

flutter run

# Improve performance (without a mobile device connected)
flutter run --release
```
